## Template

```
[Interesting fact or question]

[1–2 sentences explaining the Sunnah/hadith]

[Source: Collection + number] → Verify on Sunnah.com before posting

[Soft CTA: "Learn more in SunnahSleep" or similar]
```

## Fact-Check

**Required.** Every post must have:
- Correct collection + hadith number
