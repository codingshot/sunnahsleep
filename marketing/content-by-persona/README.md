# Content by Persona

Each folder contains marketing content tailored to that persona. Use the cross-platform workflow to adapt for all social channels.

## Persona → Content Mapping

| Persona | Primary Content | Retention Hook |
|---------|-----------------|----------------|
| **Fatima** | 5 Sunnah steps, consistency, hadith verification | Checklist completion |
| **Ahmed** | Never miss Fajr, streak tracking, alarms | Alarms set, streak |
| **Aisha** | Tahajjud made simple, Qailulah | Tahajjud alarm |
| **Yusuf** | No tracking, works offline, privacy | Trust, install |
| **Maryam** | Teach kids, verified sources, Wudu guide | Shares, Guides |

## File Naming

`[number]-[topic].md` — e.g. `01-five-sunnah-steps.md`, `02-consistency-beats-perfection.md`

## Usage

1. Pick persona for your campaign
2. Select content from that folder
3. Run fact-check if Islamic content
4. Reformat per WORKFLOW-CROSS-PLATFORM.md
5. Post to social/ folders or schedule
