# Content → Retention Mapping

Which content drives which retention signal? Use this to align campaigns with retention goals.

## Retention Goals

| Goal | Signal | Content That Drives It |
|------|--------|------------------------|
| **Activation** | First checklist item or alarm set | "5 Sunnah steps," "Never miss Fajr," "No account" |
| **Habit (3+ days)** | Return 3x in a week | 7-Day Challenge, "Track your streak" |
| **Streak** | 7+ consecutive days | "Consistency beats perfection," streak posts |
| **Alarm set** | ≥1 enabled alarm | "Fajr alarm," "Tahajjud made simple" |
| **Sleep tracking** | Log 1+ night | "Track your streak," Sleep tab feature posts |
| **PWA install** | Installed to home screen | "Works offline," "No tracking" |
| **Share** | Shared with family/friend | "Teach kids," "7-Day Challenge" |

## Campaign Flow

```
Content (persona-targeted) → Land on app → Happy path (persona-specific) → Retention signal
```

Example: "Never miss Fajr" (Ahmed) → Ahmed lands → Sets alarm → Alarm fires next day → Returns → Streak builds.
